Welcome to SquigglyNote

SquigglyNote is a modern, intuitive note-taking application designed to help you capture, organize, and manage your ideas effortlessly. Whether you're a student, professional, or creative thinker, SquigglyNote has you covered.

Key Features

📝

Easy Writing

Fast and responsive interface designed for seamless note creation without distractions.

🏷️

Smart Organization

Organize your notes with tags, categories, and custom folders for easy retrieval.

🔍

Powerful Search

Find your notes instantly with our advanced search and filtering capabilities.

☁️

Cloud Sync

Access your notes anywhere with automatic cloud synchronization across devices.

🎨

Beautiful Themes

Customize your writing experience with multiple themes and color schemes.

🔒

Secure & Private

Your notes are encrypted and stored securely with your privacy as our priority.

Get Started

Installation

Clone the repository and follow these simple steps:

git clone https://github.com/Squigglypatilkar/SquigglyNote.git cd SquigglyNote npm install npm start

Quick Start

  • Create a new note by clicking the "New Note" button
  • Start typing your thoughts and ideas
  • Organize with tags for easy categorization
  • Search and filter your notes instantly
  • Save automatically as you type

System Requirements

  • Node.js 14.0 or higher
  • npm or yarn package manager
  • Modern web browser (Chrome, Firefox, Safari, Edge)
