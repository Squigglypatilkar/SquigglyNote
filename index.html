SquigglyNote

Smart note taking made simple

About Features Getting Started GitHub

SquigglyNote is a modern, lightweight note-taking application designed for simplicity and efficiency. Capture your ideas, organize your thoughts, and access them anywhere.

Features

  • Fast and responsive interface
  • Organize notes with tags and folders
  • Powerful search and filtering
  • Cloud synchronization across devices
  • Clean, distraction-free writing experience
  • Privacy-focused and secure

Getting Started

Installation

git clone https://github.com/Squigglypatilkar/SquigglyNote.git cd SquigglyNote npm install npm start

Requirements

  • Node.js 14.0 or higher
  • npm or yarn
  • Modern web browser
